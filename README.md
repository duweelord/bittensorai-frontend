# Bittensor AI Website

## Installation

First, install the required dependencies:

```bash
npm install
# or
yarn install
# or
pnpm install
# or
bun install
```

## Development

Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

## Build

To create a production build:

```bash
npm run build
# or
yarn build
# or
pnpm build
# or
bun build
```
